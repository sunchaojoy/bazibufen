## 算法概述

1. **八字排盘整体流程** - 从出生时间到四柱八字的完整转换
2. **干支历法转换算法** - 公历转农历、干支纪年、纪日、纪时
3. **节气计算和应用** - 24节气精确计算、朔望月计算
4. **五时辰算法** - 时辰划分、子时跨日处理、时区转换
5. **大运计算方法** - 起运年龄、大运干支推算
6. **流年流月计算** - 流年干支、流月推算、流日计算
7. **神煞计算系统** - 年月日时柱神煞计算
8. **五行旺衰判断** - 令月影响、地支藏干、通根计算
9. **格局判断算法** - 正格变格、特殊格局
10. **用神选取算法** - 身强身弱、调候用神、扶抑用神
